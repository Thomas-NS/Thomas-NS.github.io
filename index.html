Projects

Planet Cube is a work-in-progress application which uses Three.js to simulate a fully functional 3D model of a rubix cube. The cube can be manipulated, randomly shuffled, automatically solved and reset to its original state.

Currently, the solving feature is a bit of a shortcut as it simply does the moves made in reverse, until it is solved. I am currently working on improving this feature by implementing a solving algorithm. I also plan to add a help button, start button, timer, improve the move counter and change the controls to be more intuitive.

This is a web-based version of the iconic Simon Says memory game using HTML, CSS and Javascript. Once you click start, your task is repeat the sequences of flashing lights which will keep getting longer and faster. You have 5 seconds to repeat the sequence each round. The number displayed on the right shows your current score and the number on the left shows your high score.

A simple interactive diary which helps track daily food groups. You can enter the date and adjust the number of servings in each pyramid section. Recommendations can be personalized by selecting the appropriate age range. Each section will display a colour depending on serving number; yellow if it is below the recommended range, green if it is within, and red if it exceeds the range.

This is a terminal-based implementation of hangman created in Java. A random word of either a chosen or random length is selected, and your goal is to guess the word letter by letter. The game ends once the ASCII hangman is completed after 7 incorrect guesses or the word is guessed correctly.
